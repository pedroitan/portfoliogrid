PP Neue Montreal is a commercial font. For demo purposes, you can use the free font 'Space Grotesk' or 'Space Mono' from Google Fonts, which is visually similar to the font used on https://valleeduhamel.com/. To use it, add the following to your _app file or <head>:

<link href="https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;700&display=swap" rel="stylesheet">

Or import in CSS:
@import url('https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;700&display=swap');

Then set fontFamily: 'Space Grotesk', 'Inter', sans-serif.
